# EduGalaxy ‚Äì Parent Dashboard

**EduGalaxy Parent Dashboard** is a web-based application designed to help parents monitor, understand, and support their child's learning journey through the EduGalaxy game. The dashboard provides insights into time spent, progress, learning trends, and personalized recommendations for each child.

This repository contains the **Parent Dashboard** frontend (React + Vite) and backend (Node.js + Express), as well as integration with the EduGalaxy game via secure APIs.

## üß© Project Overview

EduGalaxy Parent Dashboard is a **full-stack application** consisting of:
- **Frontend**: React, Vite, Framer Motion, Tailwind CSS
- **Backend**: Node.js, Express.js, Prisma, JWT Authentication, Zod validation
- **Database**: MySQL (managed via Prisma)

The dashboard provides a detailed view of the child's learning activity, weekly progress, and specific subject mastery insights.
